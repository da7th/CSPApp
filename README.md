## CSP Helper App

#### Concentrated Solar Power Class (MSc Renewable Energy at Uni. of Jordan)
#### A helper App to speed up in-class calculations

### Calculatable Values Currently Working:
* Declination
* Global Irradiance
* Direct Irradiance
* Diffuse Irradiance
* Solar Azimuth
* Surface Azimuth
* Hour Angle
* Incidence Angle
